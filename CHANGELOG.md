# Changelog

## [0.3.0] - 2026-05-02

### Новое
- Раздел «Конфиги» (/configs) для управления PersistedSettings.json — главным файлом настроек League of Legends
- Защита от перезаписи (read-only toggle) — клиент не сможет затереть настройки при выходе из игры
- Библиотека пресетов настроек: создание из текущих, применение, удаление
- Экспорт и импорт `.lolcfg` файлов для шаринга пресетов между устройствами
